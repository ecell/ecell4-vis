依存モジュールのインストール
==================================

ec4vis を動作させるには、以下のソフトウェアをインストールしておく必要があります:


- Python
- wxPython
- h5py
- numpy
- vtk

::
    # apt-get update
    # apt-get install python-wxgtk2.8
    # apt-get install python-numpy
    # apt-get install python-h5py
    # apt-get install python-vtk
